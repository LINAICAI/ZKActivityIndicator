# ZKActivityIndicator

## Requirements
Swift4
## Installation

ZKActivityIndicator is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'ZKActivityIndicator'
```

## Usage
Just like UIActivityIndicatorView
